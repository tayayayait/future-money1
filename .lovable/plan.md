
# 미래 재정 시뮬레이터 구현 계획

## 1단계: 디자인 시스템 구축

### 색상 시스템 설정
- **Primary (Deep Blue)**: 신뢰감 있는 인디고 계열 (#6366F1 메인)
- **Secondary (Teal)**: 성장과 목표를 나타내는 청록색 (#14B8A6 메인)
- **Semantic Colors**: 성공(녹색), 경고(주황), 위험(빨강), 정보(파랑)
- **시나리오별 색상**: 현재유지(회색), 조정안A/B/C 각각의 구분 색상
- **다크모드 지원**: 시스템 감지 + 수동 토글

### 타이포그래피
- **Pretendard 폰트** 적용 (한글 최적화)
- **금액 표시 전용 스타일**: 천 단위 콤마, 우측 정렬, 양/음수 색상 구분

---

## 2단계: 핵심 UI 컴포넌트

### 기본 컴포넌트
- **버튼**: 5가지 사이즈, 5가지 스타일 (primary, secondary, outline, ghost, danger)
- **입력 필드**: 일반 텍스트 + 금액 전용 입력 (자동 포맷팅)
- **카드**: 기본, 강조, 글라스모피즘, 인터랙티브 스타일
- **배지**: 상태별 색상 (성공/경고/위험/정보)
- **프로그레스 바**: 선형(예산 소진율) + 원형(목표 달성률)

### 금융 전용 컴포넌트
- **금액 표시**: +/- 부호, 색상, 천 단위 포맷
- **시나리오 카드**: 상단 컬러바, 추천 배지, 변화율 표시
- **거래 내역 아이템**: 카테고리 아이콘, 금액, 일자 표시

---

## 3단계: 차트 및 데이터 시각화

### 차트 컴포넌트 (Recharts 활용)
- **시계열 영역 차트**: 순자산 추이 (최대 4개 시나리오 비교)
- **도넛 차트**: 카테고리별 지출 비율
- **그룹 막대 차트**: 월별 수입/지출 비교
- 1년/3년/5년 기간 탭 전환
- 터치/호버 시 툴팁 표시

---

## 4단계: 백엔드 연동 (Lovable Cloud + Supabase)

### 인증 시스템
- 이메일/비밀번호 회원가입 및 로그인
- 세션 관리 및 자동 로그인 유지
- 로그아웃 기능

### 데이터베이스 테이블
- **profiles**: 사용자 기본 정보 (월 수입, 급여일, 거주형태)
- **assets**: 자산/부채 정보 (예금, 대출, 할부)
- **goals**: 재정 목표 (유형, 금액, 달성일)
- **transactions**: 수입/지출 내역 (금액, 카테고리, 날짜, 메모)
- **categories**: 지출 카테고리 마스터
- **scenarios**: 시뮬레이션 시나리오 저장

### RLS 정책
- 각 사용자는 본인 데이터만 접근 가능

## 5단계: 화면별 구현

### 1. 온보딩 플로우 (4단계)
- **Welcome**: 앱 소개 및 시작하기 버튼
- **기본 정보**: 월 실수령액, 급여일, 거주 형태 입력
- **자산/부채**: 현금/예금, 대출 잔액 (선택사항)
- **목표 설정**: 목표 유형, 금액, 달성 희망일

### 2. 메인 대시보드
- 사용자 인사 (이름 표시)
- **현금흐름 카드**: 이번 달 수입-지출 = 잔액
- **예산 소진율**: 프로그레스 바
- **목표 달성률**: 원형 프로그레스
- **인사이트 카드**: AI 기반 소비 패턴 분석
- **최근 지출 목록**: 3건 미리보기 + 더보기 링크

### 3. 하단 네비게이션
- 홈 / 내역 / 입력(메인 버튼) / 시뮬레이션 / 더보기

### 4. 빠른 입력 화면
- 큰 금액 표시 (디스플레이 크기)
- **카테고리 선택**: 12개 카테고리 그리드 (아이콘 + 색상)
- 메모 입력 (선택)
- 날짜 선택 (기본: 오늘)
- 고정비 등록 체크박스
- 숫자 키패드

### 5. 시뮬레이션 비교 화면
- 기간 선택 탭 (1년/3년/5년)
- **시계열 차트**: 현재유지 vs 조정안 비교
- **시나리오 카드 가로 스크롤**: 순자산, 비상금, 목표달성일
- **시나리오 상세**: 핵심 변화 요인, 가정 조건
- "이 시나리오로 시작하기" 버튼

### 6. 거래 내역 화면
- 월별 그룹화
- 카테고리 필터
- 스와이프로 수정/삭제

### 7. 설정 화면
- 프로필 수정
- 다크모드 토글
- 로그아웃

## 6단계: 인터랙션 & UX

### 애니메이션
- 버튼 호버: 밝기 증가 + 살짝 확대
- 카드 호버: 그림자 증가 + 살짝 올라감
- 페이지 전환: 페이드 + 슬라이드 효과
- 차트: 드로우 애니메이션 (진입 시)

### 모바일 최적화
- 하단 네비게이션 고정
- 터치 타겟 최소 44px
- 풀-투-리프레시 지원
- 바텀시트 (모달 대신)

### 접근성
- 색상 대비 4.5:1 이상
- 포커스 링 표시
- 아이콘 버튼 aria-label
- 모션 감소 설정 존중

---

## 7단계: 시뮬레이션 로직

### 핵심 계산
- 현재 소비 패턴 기반 미래 순자산 예측
- 물가 상승률, 투자 수익률 변수 적용
- 카테고리별 지출 조정 시나리오 생성
- 목표 달성 예상 시점 계산

### 시나리오 비교
- 기본(현재 유지) vs 최대 3개 조정안
- 각 시나리오별 순자산 차이 계산
- 추천 시나리오 자동 표시
